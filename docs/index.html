一站式大数据任务管理与
数据血缘可视化平台

基于 DolphinScheduler 的企业级数据门户。统一管理数据资产，智能编排任务调度，全链路追踪数据血缘，提升团队协作效率。

快速开始 查看源码

最快上手方式，包含所有依赖服务。

# 1. 克隆项目
git clone https://github.com/MingkeVan/opendataworks.git
cd opendataworks

# 2. 准备配置
cp deploy/.env.example deploy/.env

# 3. 一键启动 (前端 + 后端 + DataAgent Backend + Redis + MySQL + Portal MCP)
docker compose -f deploy/docker-compose.dev.yml up -d

适合开发调试，需手动准备基础环境。

# 后端启动（在仓库根目录）
mvn -f pom.xml -pl backend -am clean install
mvn -f pom.xml -pl backend -am spring-boot:run

# 前端启动
cd frontend
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && . "$NVM_DIR/nvm.sh"
nvm use || nvm install
npm install && npm run dev

核心特性

为数据团队打造的现代化工作台，覆盖开发、治理、监控全流程。

📊

数据资产全景

统一纳管 ODS/DWD/ADS 各层数据表。支持字段级元数管理、业务域归类，让数据资产井井有条。

🔄

智能任务调度

可视化 DAG 任务编排，深度集成 DolphinScheduler。无需硬编码，自动解析依赖，支持 SQL/Shell 多种任务类型。

🕸️

自动血缘解析

基于任务执行逻辑自动生成数据血缘图。使用 ECharts 力导向图展示，清晰追踪数据来龙去脉。

⚡

一键快速部署

提供 Docker Compose 一键启动包和离线部署方案。内置 MySQL 和 DolphinScheduler，开箱即用。

🛡️

企业级质量监控

任务执行实时监控，失败自动告警。历史日志全记录，运行状态一目了然。

🎨

现代化交互体验

采用 Vue 3 + Element Plus 构建，响应式布局。界面简洁美观，操作流畅丝滑。

技术架构

采用前后端分离架构，组件松耦合，易于扩展和维护。

+---------------------+ +----------------------+ | Frontend (Vue3) | <---> | Backend (Spring) | +---------------------+ +----------+-----------+ | | v v +---------------------+ +----------------------+ | User / Browser | | MySQL Database | +---------------------+ +----------------------+ ^ | +----------------------+ | DolphinScheduler | +----------------------+

稳健的技术栈

  • ✅ 前端: Vue 3, Vite, Element Plus, ECharts
  • ✅ 后端: Spring Boot 2.7, Mybatis Plus
  • ✅ 调度: Apache DolphinScheduler 3.2.0
  • ✅ 数据: MySQL 8.0, Flyway 迁移
  • ✅ 部署: Docker Compose, Shell Scripts
